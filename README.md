# 量化学习与研究笔记

这是一个用于记录和实现量化金融学习、研究与实践的个人代码仓库，专注于为量化实习岗位积累实战项目经验。

## 🎯 项目目标
- 系统性地学习量化交易的基础知识与方法论
- 复现经典交易策略，并进行参数优化与改进
- 构建标准化的数据处理、因子计算、回测评估工具链
- 积累可展示的高质量代码，为量化实习求职做准备

## 📁 项目结构
```
quant-tutorials/
├── data/               # 数据获取、清洗与存储
├── notebooks/          # 数据分析与研究笔记
├── strategies/         # 策略实现
├── utils/              # 工具函数库
├── requirements.txt    # 项目依赖
└── README.md           # 项目说明
```

## 🚀 技术栈
- **编程语言**: Python 3.8+
- **核心库**: pandas, numpy, matplotlib, akshare, backtrader
- **开发工具**: Git, Jupyter Notebook, VS Code

## 📦 快速开始
```bash
# 1. 将项目克隆（下载）到您的电脑
git clone https://github.com/Cherry-Bomb-1004/quant-tutorials.git

# 2. 进入项目文件夹
cd quant-tutorials

# 3. 安装项目所需的Python库
# （请先确保您已创建了requirements.txt文件）
pip install -r requirements.txt
```
